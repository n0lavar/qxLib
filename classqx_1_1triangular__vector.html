Detailed Description

template<class T>
class qx::triangular_vector< T >

Triangular vector class.

Only elements with nRow >= nCols are stored data storage example: x 0 0 0 vec[r][c] = r >= c => vec[r][c] x x 0 0 r < c => vec[c][r] x x x 0 x x x x
