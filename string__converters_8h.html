template<typename value_type >
const value_type* qx::to_char_pointer ( std::basic_string_view< value_type >  stringView)
inline

Convert string_view to the null terminated chars array.

Some libraries or even c++ types can't deal with std::string_view (for ex. std::ifstream). This function allows to reduce number of allocations, but still leads to overhead due to chars copying
