39  @brief Iterate container with filter
40  @details This function is useful when:
41  1. you don't want to use getter for container
42  (for ex. even const vector of shared_ptr will allow the client to make one more shared_ptr)
43  2. you don't want the client to know what type of container you are using
44  3. you want the client to be able to iterate only over a part of the value of an element of the vector
45  @see code examples in test_iterate.cpp
46  @tparam Container - container of elements type
47  @tparam Callable - callable type. if callable returns false, iteration breaks
48  @tparam Filter - filter type. if filter returns true, callable is applied
49  @tparam Adapter - adapter type. converts container element type to another type for callable and filter
50  @param container - container object
51  @param callable - callable object
52  @param filter - filter object
53  @param adapter - adapter object
54 **/

56  class Container,
57  class Callable,

59  class Adapter = decltype(iterate_adapters::no_change)>
60 void iterate(
