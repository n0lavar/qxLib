Detailed Description

template<class base_class_t, template< class > class smart_ptr_t, class... args_t>
class qx::reflection_creator< base_class_t, smart_ptr_t, args_t >

Reflection creator.

Performs registration of classes inherited from BaseClass and creates their instances by their ID and name. Allocates memory.
