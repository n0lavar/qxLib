Detailed Description

template<class BaseClass, template< class > class SmartPtr, class... Args>
class qx::reflection_creator< BaseClass, SmartPtr, Args >

Reflection creator.

Performs registration of classes inherited from BaseClass and creates their instances by their ID and name. Allocates memory.

Template Parameters
BaseClass- base class type
SmartPtr- smart pointer class type
