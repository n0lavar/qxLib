Detailed Description

RTTI system based on polymorphism.

Using QX_RTTI_BASE_CLASS and QX_RTTI_CLASS macros will let you to use RTTI advantages without enabling RTTI in compiler's flags common logic is taken from https://github.com/qualab/xakep-RTTI (Vladimir Kerimov, qualab)
